°❀⋆.ೃ࿔*:･ Position ･:*ೃ࿔.⋆❀°

Volver al inicio

Ideas principales

  • La propiedad de posicion especifica el tipo de metodo de posicionamiento utilizado para un elemento.
  • Los elementos se pocicionan usando las propiedades top, bottom, left, right.
  • Static: Flujo normal, sin desplazamientos.
  • Relative: Desplazamiento desde la posición original.
  • Fixed: Fijo al viewport, independientemente del scroll.
  • Absolute: Posición exacta dentro del contenedor más cercano.
  • Sticky: Se adhiere en el scroll, alternando entre relative y fixed.

Apuntes de la clase

Propiedades position: static, relative, fixed, absolut, sticky.

Static: no se ve afectado por las propiedades, top, bottom, left, right, va con el flujo normal de la pagina, esta puesta predeterminadamente.

Relative: Un elemento con position: relative; está posicionado en relación con su posición normal.

Fixed: Hace que un elemento se coloque respecto al viewport, Le dices dónde posicionarse al elemento, y se queda ahí mientras el usuario hace scroll.

Absolut: El posicionamiento absoluto se emplea para establecer de forma exacta la posición en la que se muestra en la caja de un elemento.

Sticky: Se posiciona en función de la posición de desplazamiento del usuario.

Resumen

Estas propiedades permiten controlar la disposición y comportamiento de los elementos en la página web, facilitando la creación de diseños responsivos y dinámicos.

°❀⋆.ೃ࿔*:･ Display ･:*ೃ࿔.⋆❀°

Volver al inicio

Ideas principales

  • Propiedad display: block; Siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y hacia la derecha tanto como sea posible).
  • Propiedad display: inline; Un elemento inline no comienza en una nueva línea y solo ocupa el ancho que sea necesario. Ej: span a img
  • Propiedad display: inline-block; Permite establecer un ancho y una altura en el elemento. Además, se respetan los margin y padding, que con display: inline no son respetados. No agrega un salto de línea después del elemento, por lo que el elemento puede alinearse junto a otros elementos.
  • Propiedad display: none; Se usa comúnmente en JavaScript para ocultar y mostrar elementos cuando el usuario realiza alguna acción como pulsar un botón o un enlaces sin eliminarlos ni crearlos nuevamente. Hace que el elemento no genere ninguna caja. El resultado es que el elemento desaparece por completo de la pagina y no ocupa sitio, por lo que los elementos adyacentes ocupan su lugar.
  • Propiedad display: table; Permite asignar un comportamiento de tabla a los elementos que hay dentro (anidados a la etiqueta donde estas colocando el display: table). Dentro de este elemento de pueden usar otra series de valores de display que corresponden con los roles de distintas etiquetas HTML que usas en las tablas.

Apuntes de la clase

Un elemento con display: block; siempre comienza en una nueva línea. Ocupa todo el ancho disponible del contenedor (se extiende hacia la izquierda y derecha).

Un elemento inline no comienza en una nueva línea. Solo ocupa el ancho necesario para su contenido. Los márgenes y el padding no se respetan en elementos inline

dun elemento inline-block combina características de inline y block que permite establecer un ancho y altura en el elemento. los margenes y Padding se respetan, a diferencia de inline.

El elemento display: none oculta un elemento completamente del flujo del documento.

Un elemento display: table asigna un comportamiento de tabla al elemento y los elementos hijos se comportan como celdas de una tabla.

Resumen

Las propiedades de display son fundamentales en CSS, ya que determinan cómo se comportan y presentan los elementos en una página web. En resumen, entender y utilizar correctamente las propiedades de display es esencial para cualquier desarrollador web que busque crear diseños efectivos, accesibles y atractivos.

°❀⋆.ೃ࿔*:･ Animation ･:*ೃ࿔.⋆❀°

Volver al inicio

Ideas principales

  • CSS permite animar la mayoria de los elementos HTML sin usar javaScript.
  • Cuando especifica estilos CSS dentro de la regla @keyframes, la animacion cambiara del estilo actual al estilo nuevo.
  • Para que una animacion funcione, se debe vincular la animacion a un elemento.
  • animation-duration: especifica cuanto tiempo debe transcurrir una animacion para completar un ciclo.
  • animation-iteration-count: especifica el numero de veces que se debe reproducir una animacion. se expresa con numeros cerrados y se puede usar tambien un infinito.
  • animation-delay: especifica un retraso para el inicio de una animacion. Se expresa en sefundos. Se pueden utilizar unidades negativas.
  • animation-play-state: especifica si la animacion se esta ejecutando o esta en pausa.
  • animation-timing-function: especifica la curva de velocidad de la animacion.
    • linear: especifica una animacion con la misma velocidad de principio a fin.
    • ease: especifica una animacion con un inicio lento, luego rapido y luego termina lentamente (por defecto)
    • ease-in: especifica una animacion con un inicio lento.
    • ease-out: especifica una animacion con un final lento.
    • ease-in-out: especifica una animacion con un inicio y un final lentos.
    • cubic-bezier(n, n, n, n): permite definir sus propios valores en una funcion cubic-bezier

Apuntes de la clase

Sintaxis

@keyframes-> Regla @keyframes, "nombre"-> Nombre de la animacion, from{ }-> Estilo actual, to{ }-> Estilo nuevo

Sintaxis por porcentaje

@keyframes-> Regla @keyframes, "nombre"-> Nombre de la animacion, 0%{ }-> Estilo actual, 50%{ }-> Estilo nuevo al 50%, 100%{ }-> Estilo nuevo al 100%.

Resumen

En resumen, las animaciones en CSS mejoran la interacción y la apariencia de las páginas web, haciendo que sean más atractivas y funcionales.

°❀⋆.ೃ࿔*:･ Nuevas propiedades CSS ･:*ೃ࿔.⋆❀°

Volver al inicio

Ideas principales

  • Es un intento por reducir el uso de codigo javaScript y para estandarizar funciones populares, CSS3 no solo cubre diseño y estilos web sino tambien forma y movimiento.
  • Border-radius corner-radius: esta propiedad genera esquinas redondeadas para la caja formada por el elemento, posee dos parametros diferentes que dan forma a la esquina (horizontal y vertical) se puede llegar a crear hasta un elipse.
  • Box-shadow: esta propiedad crea sombras, puede tomar cinco parametros (el color, el desplazamiento horizontal/vertical, valor de difuminacion y un inset para una sombra mas intensa).
  • @font-face: esta regla nos permite cargar cualquier fuente que necesitemos.
  • linear-gradient: esta funcion puede ser aplicada a las propiedades background-image para generr un gradiente lineal.
  • RGBA: esta funcion es una mejora de rgb() con un canal alfa que especifica la opacidad de un color.
  • HSLA: los valores de color HSLA son una extension de los valores de color HSL con un canal alfa, que especifica la opacidad de un color.
  • Outline: esta propiedad fue mejorada con la incorporacion de otra llamada outline-offset. ambas generan un segundo borde alejado del borde original del elemento.
  • Transform: esta propiedad modifica la forma de un elemento, utiliza cuatro funciones basicas (scale, rotate, skew y translate).

Apuntes de la clase

Este nivel de cambio convierte CSS3 en una tecnologia practicamente inedita comparada con versiones anteriores.

Para algunas propiedades se deben usar los prefijos -moz-, -o-, y -webkit- para que funcionen en navegadores basados en motores Gecko y WebKit, como Firefox, Safari y google Chrome.

Podemos crear desde esquinas redondeadas y sombras hasta transformaciones y repposicionamiento de los elementos ya presentados en pantalla, cada posible efecto aplicado previamente utilizando javaScript fue cubierto.

border-radius: Npx/Npx;

box-shadow: #000000 Npx Npx Npx inset;

linear-gradient: (top, #FFFFFF 50%, #000000 90%);

background-color: rgba(255, 0, 0, 0.3);

background-color: hsla(120, 100%, 50%, 0.3);

outline: 1px solid #000000;

outline-offset: 10px;

Resumen

Estas características de CSS3 enriquecen el diseño web, permitiendo crear interfaces más interactivas y estéticamente agradables sin depender tanto de JavaScript.

°❀⋆.ೃ࿔*:･ Referencias avanzadas ･:*ೃ࿔.⋆❀°

Volver al inicio

Ideas principales

  • Atributo id: es como un nombre que identifica al elemento. el valor de este atributo no puede ser duplicado. este nombre debe ser unico en todo el documento.
  • Atributo class: este atributo es mas flexible y puede ser asignado a varios elementos HTML en el documento que comparten un diseño similar.
  • Elementos: un selector de elementos en CSS es una regla que selecciona todos los elementos de un tipo especifico en el documento HTML. Es el selector mas basico y se refiere directamente al nombre de una etiqueta HTML.
  • Pseudoclases - :nth-child(): nos permite encontrar un hijo especifico.
  • Pseudoclases - :first-child: existen otras importantes pseudoclases relacionadas con esta ultima. hace referencia solo el primer hijo. no requieren palabras clave o parametros.
  • Pseudoclases - :last-child: referencia solo el ultimo hijo. no requieren palabras clave o parametros.
  • Pseudoclases - :only-child: este afecta un elemento siempre y cuando sea el unico hijo disponible. no requieren palabras clave o parametros.
  • Pseudoclases - :not(): nos permite declarar una excepcion. los estilos en la regla creada con est pseudoclase seran asignados a todo elemento excepto aquellos incluidos en la referencia entre parentesis.
  • Selector de atributo: selecciona elementos HTML en funcion de la presencia o el valor de uno de sus atributos. Esto permite aplicar estilos a los elementos que tienen un atributo especifico, o cuyo atributo tiene un valor determinado.

Apuntes de la clase

El atributo id se utiliza para identificar de manera única a un elemento en un documento HTML.

El atributo class permite agrupar varios elementos que comparten un diseño o comportamiento similar. Su flexibilidad permite asignar el mismo valor de class a múltiples elementos, lo que facilita la aplicación de estilos coherentes a través de diferentes partes de un documento.

Los selectores de elementos en CSS permiten seleccionar todos los elementos de un tipo específico en el documento HTML.

Las pseudoclases son selectores que permiten aplicar estilos a elementos en función de su estado o posición en el documento.

El selector de atributo permite aplicar estilos a elementos basándose en la presencia o valor de uno de sus atributos.

Resumen

Los atributos id y class, junto con los selectores y pseudoclases, son herramientas esenciales en CSS para identificar y aplicar estilos a los elementos en un documento HTML. Estos conceptos permiten una mayor flexibilidad y control en el diseño web.
